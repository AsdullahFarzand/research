## Citation Format Guide

### IEEE Citation Format
```
[Author(s)], "[Title]," [Journal/Conference], vol. [volume], no. [number], 
pp. [pages], [Month] [Year]. doi: [DOI]
```

**Example:**
```
B. A. Alahmadi et al., "99% False Positives: A Qualitative Study of SOC 
Analysts' Perspectives on Security Alarms," in Proc. USENIX Security 
Symposium, 2022, pp. 2639-2656.
```

### APA Citation Format (7th Edition)
```
Author, A. A., & Author, B. B. (Year). Title of article. Title of Periodical, 
volume(issue), page–page. https://doi.org/xxxxx
```

**Example:**
```
Alahmadi, B. A., Axon, L., & Sheratt, E. (2022). 99% false positives: 
A qualitative study of SOC analysts' perspectives on security alarms. 
Proceedings of the USENIX Security Symposium, 2639-2656.
```

### Chicago Citation Format
```
Author Last Name, First Name. "Article Title." Journal Title Volume, 
no. Issue (Year): Page–Page. DOI.
```

### Harvard Citation Format
```
Author(s) (Year) 'Title of article', Journal Title, Volume(Issue), 
pp. page-page. doi: DOI.
```

### arXiv Preprints (2xxx.xxxxx format)
- Pre-publication research papers
- May not be peer-reviewed
- Check for published versions before citing

- Springer journal articles
- Peer-reviewed publications

- Specific venue and year required for citation
